## 🧠 Model Comparison

Two architectures were trained and evaluated on FashionMNIST:

| Metric | DNN | CNN |
|--------|-----|-----|
| **Type** | Fully-connected (512→256→128) | 3 Conv blocks + FC |
| **Parameters** | ~670K | ~1.2M |
| **Test Accuracy** | ~89% | **~92%** ✅ |
| **Training Time** | ~30s/epoch (CPU) | ~90s/epoch (CPU) |

**Winner: CNN** — better accuracy due to spatial feature extraction via convolutions.

|---|---|---|---|---|---|---|---|---|---|

## 🚀 Quick Start (Local)

### Prerequisites
- Python 3.11+
- Docker (for containerized run)

### Option 1: Run with Python directly

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train the model (downloads FashionMNIST automatically)
python train.py

# 3. Start the API server
python main.py
# or:
uvicorn main:app --host 0.0.0.0 --port 8080

# 4. Test it
python test_api.py
```

### Option 2: Run with Docker

```bash
# 1. Train the model (if not done already)
python train.py

# 2. Build Docker image
docker build -t fashion-mnist-api .

# 3. Run container
docker run -p 8080:8080 \
    -e API_KEY="my-secret-key" \
    fashion-mnist-api

# 4. Test
curl http://localhost:8080/health
```

## 📡 API Documentation

Once running, open interactive docs at: **http://localhost:8080/docs**

### Endpoints

#### `GET /` — API Info
```bash
curl http://localhost:8080/
```

#### `GET /health` — Health Check
```bash
curl http://localhost:8080/health
```
Response:
```json
{
  "status": "healthy",
  "model_loaded": true,
  "model_type": "CNN",
  "device": "cpu"
}
```

#### `POST /predict` — Make Prediction
**Auth required**: `X-API-Key` header

```bash
curl -X POST http://localhost:8080/predict \
    -H "X-API-Key: fashion-mnist-api-key-2026" \
    -H "Content-Type: application/json" \
    -d '{"image": [0,0,0,...,255,...,128]}'
```

**Input**: JSON with `image` field — a flat array of 784 float values (0–255, 28×28 grayscale).

**Output**:
```json
{
  "predicted_class": 3,
  "class_name": "Dress",
  "confidence": 0.9521,
  "all_probabilities": [0.001, 0.002, 0.005, 0.952, 0.010, 0.008, 0.012, 0.003, 0.004, 0.003],
  "model_type": "CNN"
}
```

### Authentication
- Set via `API_KEY` environment variable (default: `fashion-mnist-api-key-2026`)
- Pass as `X-API-Key` header in requests
- Requests without valid key return `401 Unauthorized`

## ☁️ Deployment (GCP Cloud Run)

Full guide: [`GCP_DEPLOY.md`](GCP_DEPLOY.md)

Quick steps:

```bash
# 1. Set up GCP
gcloud auth login
gcloud config set project YOUR_PROJECT_ID

# 2. Enable APIs
gcloud services enable run.googleapis.com artifactregistry.googleapis.com

# 3. Create Artifact Registry
gcloud artifacts repositories create fashion-mnist-repo \
    --repository-format=docker --location=asia-east1

# 4. Auth Docker
gcloud auth configure-docker asia-east1-docker.pkg.dev

# 5. Build, tag & push
docker build -t fashion-mnist-api .
docker tag fashion-mnist-api asia-east1-docker.pkg.dev/$PROJECT_ID/fashion-mnist-repo/fashion-mnist-api:v1
docker push asia-east1-docker.pkg.dev/$PROJECT_ID/fashion-mnist-repo/fashion-mnist-api:v1

# 6. Deploy to Cloud Run
gcloud run deploy fashion-mnist-api \
    --image=asia-east1-docker.pkg.dev/$PROJECT_ID/fashion-mnist-repo/fashion-mnist-api:v1 \
    --platform=managed --region=asia-east1 --allow-unauthenticated \
    --memory=1Gi --set-env-vars="API_KEY=your-key" --port=8080
```

## 📊 Training Details

- **Preprocessing**: Normalize pixel values using computed mean/std from training set
- **DNN**: 3 hidden layers (512, 256, 128) with BatchNorm, ReLU, Dropout(0.3)
- **CNN**: 3 conv blocks (32→64→128 filters) with BatchNorm, MaxPool, Dropout → FC(256→10)
- **Optimizer**: Adam (lr=0.001)
- **Scheduler**: ReduceLROnPlateau (patience=2, factor=0.5)
- **Loss**: Cross-Entropy
- **Epochs**: 10

Run training: `python train.py`
